The Style Guide

A Living Document of the IGN web styles.

The essence of this guide is to break down all of the components we use across the IGN experience into atomic chunks that can be defined and documented in such a manner that they become reusable across projects. This begins with defining a default style for all basic web components that can be referenced at the beginning of each project for both designers and engineers. There are a great many benefits to this process but the main one I will mention here is how it will greatly speed up our process when we can reference predefined styles instead of redefining a basic style for each project.

This is only the first iteration of this project and it should be treated as a living document where we will consistently add and refine the contents.
